# Saoas_Toolbox
 - **Installation:**
 - Just run the following code in Powershell!
 - To get access contact and send me the license_request file created with the Toolbox! 
```
irm https://toolbox.saoas.ovh/get | iex
```
 - **Uninstall:**
 - Just run the following code in Powershell!
```
irm https://toolbox.saoas.ovh/remove | iex
```
